常见前端语言和数据库搭配

2025-03-10 · AuranLu

数据库：Oracle 大型数据库、Access、SQL Server（ms SQL）、MySQL 常用中小型数据库 中间件：IIS、Apache、Nginx、Tomcat 脚本：PHP、ASP、ASPX（ASP.NET）、JSP

脚本和数据库的搭配： PHP + MySQL ASP + Access 、ASP + SQL Server（ms SQL） ASPX（ASP.NET）+ SQL Server（ms SQL） JSP + MySQL 、JSP + Oracle

综合搭配： Linux + Apache + PHP + MySQL Linux + Nginx + PHP + MySQL Windows + IIS + ASPX（ASP.NET）+ SQL Server（ms SQL） Linux + Tomcatt + JSP + MySQL Linux + Tomcat + JSP + Oracle

常见脚本语言及其后缀名格式

ASP：.asp

ASPX：.aspx

JAVASCRIPT：.js

PHP：.php


常见数据库及其后缀名格式

Access: Access数据库的文件后缀是.mdb

MSSQL: MSSQL数据库的主要数据文件后缀是.mdf，附加数据文件后缀是.ndf，日志文件后缀是.ldf

MySQL: MySQL数据库的文件后缀是.myd，导出文件通常是.sql

Paradox: Paradox数据库的文件后缀是.db

Oracle: Oracle数据库的文件后缀是.dbf，导出文件后缀是.dmp，日志文件后缀是.log

dBase: dBase数据库的文件后缀是.dbf

FoxPro: FoxPro数据库的文件后缀是.dbf

MS Works: MS Works数据库的文件后缀是.wdb

MongoDB: MongoDB数据库的文件后缀是.bson

PostgreSQL: PostgreSQL数据库的文件后缀包括.backup、.sql和.txt

SQLite: SQLite数据库的文件后缀是.db，也常使用.sqlite和.bak

其他常见文件格式，除了上述数据库文件后缀，还有一些常见的数据文件格式，如：

CSV: 逗号分隔的值文件。

DAT: 数据文件。

DB: Borland的Paradox 7表
