1. Profile（个人资料）#

yaml复制编辑# 头像路径
avatar: /avatar/Misaka.jpg
# 作者名称
author: AuranLu
# 个性签名
signature: witness me
  • avatar：指定头像的图片路径，如果为空，某些主题可能不显示默认头像。
  • author：网站作者的名字，通常不会影响 Hexo 运行，但部分主题可能需要。
  • signature：个性签名，通常出现在个人资料区域。

📌 建议

  • 避免空值，如果 avatar: ""，可能导致头像无法加载。
  • author 和 signature 为空时不会报错，但主题可能不会显示对应内容。

3. Search（搜索功能）#

yaml复制编辑search:
  enable: false
  url: 
  engine: bing
  • enable: false 关闭搜索功能。
  • url: 必须填写，否则某些搜索插件可能会报错。
  • engine: bing 指定搜索引擎（可选 google、baidu）。

📌 建议

  • 如果 search.enable: true，url 不能为空，否则搜索框可能无法使用。

4. SEO（搜索引擎优化）#

yaml复制编辑SEO_title: fi3ework's Studio
SEO_keywords:
  - hexo
  - 'hexo-theme'
  - 'hexo-blog'
  • SEO_title 是网站的标题，用于 SEO 优化。
  • SEO_keywords 是关键词数组，不建议为空，否则 SEO 可能受影响。

📌 建议

  • 如果 SEO_keywords 为空，建议使用 SEO_keywords: []，避免解析错误。

5. 主题设置#

yaml复制编辑# 文章摘要长度
truncate_length: 300
# 是否显示“阅读更多”按钮
read_more_button: false
# 是否启用目录
toc: true
# 文章阅读信息（字数 & 阅读时间）
reading_info: true
  • truncate_length: 300 控制首页摘要长度，0 表示不显示摘要。
  • read_more_button: false 关闭“阅读更多”按钮。
  • toc: true 启用文章目录（Table of Contents）。
  • reading_info: true 启用字数统计和阅读时间。

📌 建议

  • 这些参数可以为空，Hexo 不会报错，但 truncate_length: 建议填 0 或 300，避免前端解析异常。

7. 统计分析#

yaml复制编辑busuanzi: true
busuanzi_pv_or_uv: 'pv'
busuanzi_slug: 'PV: ${count} :)'
baidu_analytics: 
google_analytics: 
CNZZ_analytics:
  • busuanzi: true 开启不蒜子统计（访客统计）。
  • busuanzi_pv_or_uv: 'pv' 统计 pv（页面浏览量）或 uv（独立访客）。
  • baidu_analytics:、google_analytics:、CNZZ_analytics: 为空时不会报错，但需要填写正确 ID 才能使用相应分析功能。

📌 建议

  • 如果不使用 Google Analytics，则 google_analytics: 可留空，不会影响 Hexo 运行。

8. 其他设置#

yaml复制编辑favicon: /assets/favicon.ico
copyright:
  enable: true
  license: '本文采用<a rel="license" href="http://creativecommons.org/licenses/by-nc/4.0/">知识共享署名-非商业性使用 4.0 国际许可协议</a>进行许可'
  • favicon 是浏览器图标路径，留空可能导致部分主题报错（建议提供默认图标）。
  • copyright.enable: true 开启版权声明，建议不留空。

9. 数学公式 & 图表#

yaml复制编辑math:
  mathjax:
    enable: false
    version: 3.2.0
  • mathjax.enable: false 关闭数学公式渲染。
  • version: 3.2.0 指定 MathJax 版本。

📌 建议

  • 如果 mathjax.enable: true，确保 version 不为空，否则可能导致 MathJax 加载失败。
